链表回文检测

Java 数据结构 2020/02/27

1 会改变原结构，时间复杂度O(n)，空间复杂度O(1)

思路

  1. 将前半部分翻转并找到中点
  2. 分两指针以中间节点为中心分别向前和向后移动
  3. 依次比较两指针的节点是否相等，若有节点不等，则为非回文链

2 不会改变原结构，时间复杂度O(n)，空间复杂度O(n)

思路

  1. 前半部分缓存在栈中
  2. 分两指针以中间节点为中心分别向前（栈）和向后（原链表）移动
  3. 依次比较两指针的节点是否相等，若有节点不等，则为非回文链
