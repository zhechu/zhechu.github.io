快速排序

Java 算法 2020/02/23

原理：先从数列中取出一个数作为基准数，分区过程，将比这个数大的数全放到它的右边，小于或等于它的数全放到它的左边，再对左右区间重复第二步，直到各区间只有一个数

平均时间复杂度 额外空间复杂度 稳定性 备注
O(nlogn) O(1) 不稳定 n 大时较好
