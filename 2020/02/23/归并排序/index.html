归并排序

Java 算法 2020/02/23

原理：将已有序的子序列合并，得到完全有序的序列；即先使每个子序列有序，再使子序列段间有序

平均时间复杂度 额外空间复杂度 稳定性 备注
O(nlogn) O(n) 稳定 n 大时较好
