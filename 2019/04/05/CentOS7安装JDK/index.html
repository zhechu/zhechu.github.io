CentOS7安装JDK

Linux CentOS7 2019/04/05

卸载 JDK

若已装有 JDK，需先卸载

安装 JDK-1.8

通过 RPM 方式安装 JDK-1.8

配置环境变量

使环境变量生效（对于当前 Session 才生效）

重启主机使环境变量对所有 Session 都生效

安装 JDK-1.7

将 RPM 包改为 jdk-8u11-linux-x64.rpm 即可。其它步骤与安装 JDK-1.8 一样。
