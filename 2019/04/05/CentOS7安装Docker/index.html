1.3 安装

要求内核版本不低于 3.10

添加 Docker 源

查看 docker-engine 的所有版本,找到期望版本的精确名字,如：1.13.1-1.el7.centos

查看某个docker-engine版本的依赖，如 1.13.1 依赖 docker-engine-selinux， 版本>= 1.13.1-1.el7.centos

由于 docker-engine 会默认安装 docker-engine-selinux 13，而安装 docker-engine-selinux 13 需要将本机的 selinux 升级，所以手动安装 docker-engine-selinux 13

1.4 配置加速器

1.4.1 官方加速器

默认的镜像拉取是从 Docker 官方镜像库下载，速度较慢。配置国内官方镜像加速器，可以提高拉取镜像效率

需重启 Docker 才会生效

1.4.2 DaoCloud 加速器

DaoCloud 加速器配置文档

https://www.daocloud.io/mirror

Docker 官方加速器不是很稳定，推荐使用 DaoCloud 的

需重启 Docker 才会生效
