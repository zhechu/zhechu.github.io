1 指定扫描的包

若启动类只使用 @SpringBootApplication 且没有加扫描参数。则其会扫描启动类所在包及所在包目录下的所有文件。因此，指定扫描的包，会减少 Spring Boot 启动时扫描的文件，从而提高 Spring Boot 启动效率。
